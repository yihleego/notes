13. 以下两种写法哪一个符合 PEP8？为什么？

```
spam(ham[1], {eggs: 2})
spam( ham[ 1 ], { eggs : 2 } )
```
第一种写法符合 PEP8，因为它遵守了括号、方括号、花括号内不留多余空格，以及字典冒号左右空格的规范。
